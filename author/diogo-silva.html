How to get unique values in an Elm list?

 qui 28 janeiro 2021  —  Diogo Silva

You do it like this:

unique : List a -> List a
unique l =
    let
        incUnique : a -> List a -> List a
        incUnique elem lst =
            case List.member elem lst of
                True -> lst
               False -> elem :: lst
    in
        List.foldr incUnique [] l

There you go. Have fun!
